# Snap Magic Components Collection

This repository contains a curated collection of KiCad components sourced from Snap Magic for personal use.

## About Snap Magic

[Snap Magic](https://www.snapmagic.com/) is a comprehensive electronic component search engine and library management platform that provides engineers with access to millions of verified component symbols, footprints, and 3D models. The components in this repository have been obtained from the Snap Magic library.

## About This Repository

This repository is a **submodule** of the larger [my_kicad_mods](https://github.com/HenraL/my_kicad_mods.git) project, which serves as a collection of various KiCad component libraries organized as Git submodules.

### Adding as a Submodule

```bash

git submodule update --init --recursive
```
